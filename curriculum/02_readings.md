# Foundry90 – Weekly Reading List

This is a curated list of high-signal resources for each week.

Use it as a menu, not a prison. If you know something already, skim and move on.

---

## Week 1 – Next.js + TypeScript

- Next.js App Router docs  
  https://nextjs.org/docs/app/building-your-application
- Routing & layouts  
  https://nextjs.org/docs/app/building-your-application/routing
- Rendering & data fetching  
  https://nextjs.org/docs/app/building-your-application/rendering
- TypeScript Handbook (intro)  
  https://www.typescriptlang.org/docs/handbook/intro.html
- Tailwind CSS basics  
  https://tailwindcss.com/docs/utility-first

**If you only read one thing:** Next.js “Building Your Application” section.

---

## Week 2 – Rust + Axum + Async

- The Rust Book (chapters 4–10 are plenty to start)  
  https://doc.rust-lang.org/book/
- Tokio tutorial  
  https://tokio.rs/tokio/tutorial
- Axum examples  
  https://github.com/tokio-rs/axum/tree/main/examples
- Error handling with `thiserror`  
  https://docs.rs/thiserror/latest/thiserror/
- Serde overview  
  https://serde.rs/

  https://websockets.readthedocs.io/en/stable/intro.html

## Week 8 – Observability

- Google SRE – Monitoring distributed systems

- AWS CloudWatch Logs & Metrics docs (for whichever services you use)

---

## Weeks 9–12 – Systems, Rust, Next.js, SwiftUI

At this point, prioritize what you’re actually building:

- Axum advanced topics (middleware, extractors, shared state)
- React Server Components concepts / App Router data fetching patterns
- Apple SwiftUI tutorials  
  https://developer.apple.com/tutorials/swiftui
- High-level system design articles (ByteByteGo and similar)

Don't try to read everything; read what unblocks the thing you're currently building.
