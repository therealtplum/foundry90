# F90Shared

Shared Swift package containing models and services used by both the macOS (FMHubControl) and iOS (F90Mobile) applications.

## Contents

- **KalshiModels.swift**: Data models for Kalshi markets, accounts, positions, and credentials
- **KalshiService.swift**: Service layer for interacting with the Kalshi API backend

## Usage

This package can be added as a local Swift Package Manager dependency to both the macOS and iOS projects.
