# Foundry90 Docs

This folder contains documentation and assets that support the curriculum and capstones.

It is the home for:

- architecture diagrams  
- sequence diagrams  
- PNG/SVG exports  
- Mermaid `.mmd` sources  
- branding assets (logos, color palettes, typography)  
- PDFs  
- presentation slides  
- future GitHub Pages site content
